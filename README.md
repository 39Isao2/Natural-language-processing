# 自然言語処理について

## 自然言語処理とは？
そもそも「自然言語」とは私たちが普段話している日本語や英語などの言語のことです。<br>
具体的な処理ですが、まず形態素解析といって文章を可能な限り小さい単位（名詞、動詞、助詞など）に分割することから始まり、次にかがり受けなどの構造を明らかにする構文解析を行います。<br>
そして最後に文の中で「誰がいつどこで何をした」という5W1Hを意味解析で判定します。<br>
